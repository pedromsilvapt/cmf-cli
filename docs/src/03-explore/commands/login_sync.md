# login sync

<!-- BEGIN USAGE -->

### Description

Name | Description
---- | -----------
`Sync credentials from the .cmf-auth.json file into each specific tool (npm, nuget, docker, etc...) configuration files.

Usage
-----

```
cmf login [<repositoryType> [<repository>]] sync [options]
```

### Arguments

Name | Description
---- | -----------
`<CIFS|Docker|NPM|NuGet|Portal>` | Type of repository for login (values: portal, docker, npm, nuget, cifs)
`<repository>` | URL of repository for login

### Options

Name | Description
---- | -----------
`-?, -h, --help` | Show help and usage information
